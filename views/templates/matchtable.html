Template für Spielplan

Einige vorhandene Marker als Beispiel:

###MATCH### Subpart für ein Spiel
###MATCH_ROLL### Marker für den Typ einer Zeile. Dies ist eine laufende Nummer bei 0 beginnend und leaguetable.table.rowroll endend.
###MATCH_POSITION_MARK### Marker für spezielle Markierung einer Tabellenposition aus Wettbewerbsdatensatz
###MATCH_POSITION### Marker für Tabellenplatz
###MATCH_HOME### Marker für UID des Heimteams
###MATCH_HOME_NAME### Marker für Name des Heimteams
###MATCH_HOME_SHORT_NAME### Marker für Kurzname des Heimteams
###MATCH_HOME_LINK### Subpart für Link auf das Heimteam
###MATCH_HOME_LOGO### Marker für das Logo des Heimteams
###MATCHCOUNT### Marker für Anzahl der Spiele
###MATCH_COMPETITION_NAME### Marker für Name des Wettbewerbs
###MATCH_DATE### Marker für Datum des Spiels
###MATCH_GOALS_HOME### Marker für Tore des Heimteams nach Spielende
###MATCH_GOALS_GUEST### Marker für Tore des Gastteams nach Spielende

Weitere Marker ergeben sich aus den obigen Beispielen. Statt HOME verwendet man z.B. GUEST für die Gastmannschaft.
